# greenroom

## Setup
1. Add `OPENAI_API_KEY` and `PUSHER_*` env vars in Vercel
2. Drop `.txt` files into `.rag-docs/` for RAG mode
3. TV: `/tv.html` — Mobile: `/mobile.html`

## Files needed in repo root
- `pusher.min.js` — download from https://js.pusher.com/8.2.0/pusher.min.js
- `qrcode.min.js` — download from https://cdnjs.cloudflare.com/ajax/libs/qrcodejs/1.0.0/qrcode.min.js
